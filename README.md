# TrackYourBudget

A full-stack AI-driven personal finance tracker that helps users manage expenses, set savings goals, and import transactions via CSV/PDF.

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn-ui
- **Backend:** Supabase (Auth, Database, Edge Functions)
- **AI:** OpenAI API (Budget Advisor)
- **Deployment:** Vercel

## Features

- 📊 **Expense Tracking** — Add, edit, and delete expenses with categories and payment methods
- 🤖 **AI Budget Advisor** — Get personalized financial insights powered by AI
- 🎯 **Savings Goals** — Set goals with deadlines and auto-calculated monthly targets
- 📥 **CSV/PDF Import** — Import bank statements and transaction files directly
- 🔐 **Google Sign-In** — Secure OAuth authentication
- 📈 **Real-time Dashboard** — Visual spending breakdowns with charts and analytics
- 💰 **Budget Management** — Set monthly and category-level budget limits
- 🔗 **Shared Access** — Share your dashboard with others via access codes

## Getting Started

```sh
git clone https://github.com/your-username/trackurbudget.git
cd trackurbudget
npm install
npm run dev
```
